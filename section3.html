Principaux services d'internet

Le courrier électronique

Le courrier électronique est un service de transmission de messages envoyés électroniquement via Internet dans la boîte aux lettres électronique d'un destinataire choisi par l'émetteur.

Les principaux protocoles de service e-mail sont :

  • Les protocoles "Sortants" qui permettent de gérer la transmission du courrier entre les serveurs.
  • Les protocoles "Entrants" : Ce sont des protocoles de réception. Deux protocoles sont utilisés : POP ou IMAP.
    • POP (Post Office Protocol) et IMAP (Internet Message Access Protocol) assurent la réception, la consultation des messages et l'authentification.

Problèmes liés aux e-mails :

  • Incertitude quant à l'acheminement du courriel à bon port.
  • Le problème du courrier indésirable (pourriel).

Transfert des fichiers FTP

FTP est un protocole de communication destiné à l'échange de fichiers sur un réseau TCP/IP. Il permet de :

  • Copier des fichiers vers un autre ordinateur du réseau.
  • Administrer un site web et de supprimer ou de modifier des fichiers sur cet ordinateur.

On distingue deux sens de transfert :

  • Le téléchargement (download) : consiste à transférer des fichiers d'une machine distante (serveur ftp) à une machine cliente.
  • L'hébergement (upload) : permet de transférer des fichiers d'une machine cliente à une machine distante (serveur ftp).

L'accès à des systèmes distants

Telnet (TErminal NETwork) est un protocole réseau utilisé sur tout réseau supportant le protocole TCP/IP. Il peut se connecter à n'importe quelle machine qui dispose d'un service de Telnet.
