Chapitre 2 : principes d'internet

Définition réseau informatique :

Ensemble de périphériques ou d'ordinateurs reliés via un câble qui permet l'échange et la transmission des données.

Avantages : Le partage des fichiers et du matériel, ce qui permet une réduction des coûts et du temps.

Principaux types de réseaux :

  1. LAN (local) :
    • Limité géographiquement.
    • Topologie : bus, étoile, anneau.
    • Technologie : câbles à paires torsadées, wifi.
  2. MAN :
    • Relie plusieurs LAN géographiquement proches pour faciliter la communication.
    • Topologie : bus, étoile en général.
    • Aire géographique : 1 à 100 km.
    • Nombre de stations : plus de 1000 machines.
  3. WAN :
    • Interconnecte plusieurs réseaux LAN géographiquement éloignés.
    • Aire géographique : toute la surface de la terre.
    • Topologie : bus, étoile en général.
    • Technologie : fibre optique.

Topologies des réseaux :

Topologie en bus :

  • Tous les nœuds sont reliés par un conduit unique.
  • Lorsqu'une station émet des données, elles circulent sur toute la longueur du bus.

Avantages :

  • Nécessite moins de câblage.
  • Facile à mettre en place.
  • Si une station tombe en panne, le reste du réseau ne sera pas perturbé.

Inconvénients :

  • La rupture du câble implique un dysfonctionnement du réseau.

Topologie en anneau :

  • Les ordinateurs sont reliés entre eux pour former un anneau.
  • Un jeton circule autour de l'anneau, la station qui a le jeton émet les données lorsque les données reviennent, le jeton passe vers la station voisine.

Avantages :

  • Aucune collision.
  • La panne d'un port nuit à l'ensemble du réseau.

Topologie en étoile :

  • La communication entre les stations du réseau se fait par l'intermédiaire d'un élément central.
  • Les données passent par un hub qui peut les envoyer à toutes les machines, ou par un switch qui les transmet uniquement aux destinataires.

Avantages :

  • Une panne d'un port n'affecte pas l'ensemble du réseau.
  • Plus de câblage comparé aux autres topologies.
  • Dépendance aux concentrateurs : si un concentrateur tombe en panne, les ordinateurs ne peuvent plus communiquer.

Topologie point à point (ou maillée) :

  • Utilisée dans les réseaux de distribution (exemple : internet)
  • L'information parcourt le réseau en suivant des itinéraires divers.

Inconvénients :

  • Le nombre de liaisons nécessaires devient très élevé.

Principaux composants d'interconnexion :

  • Carte réseau : Constitue l'interface physique entre l'ordinateur et le support de communication.
  • Concentrateur (hub) : Équipement physique avec plusieurs ports servant à relier plusieurs ordinateurs. Il dirige les données et les partage avec l'ensemble des machines connectées à lui.
  • Commutateur (switch) : Partage les données uniquement avec le destinataire de ces données.
  • Passerelle : Dispositif servant à connecter des réseaux de différentes architectures de protocoles. Exemple : internet et réseau local.
  • Routeur : Matériel de communication de réseau informatique servant à diriger l'information dans la direction appropriée de manière optimale.
  • Modem : Lie le réseau téléphonique aux réseaux informatiques. Il convertit le signal numérique en signal analogique et vice versa.
  • Répéteur : Régénère un signal affaibli pendant le transport, ce qui permet d'étendre la distance maximale entre deux nœuds d'un réseau.

Protocoles de communication :

Les protocoles de communication sont les consignes obligatoires à respecter pour la communication entre les réseaux.

Les plus utilisés sont : HTTP, FTP, SMTP, POP. La plupart d'entre eux sont bas és sur le protocole TCP/IP (Transmission Control Protocol/Internet Protocol).

TCP/IP : Il représente les règles de communication sur internet et repose sur la notion d'adresse IP.

L'adresse IP est une adresse unique attribuée à chaque ordinateur sur internet, elle permet d'acheminer les données à la bonne destination. Les adresses IP sont généralement notées avec quatre champs numériques compris entre 0 et 255, séparés par des points. La version IPv4 est encore actuellement la plus utilisée.

Adresse IP version 4 (IPv4) : Les adresses sont codées sur 32 bits (2^32 = 4 294 967 296). Les adresses sont généralement notées avec quatre champs numériques compris entre 0 et 255, séparés par des points. La version IPv4 est encore actuellement la plus utilisée.

Représentation des adresses IPv6 : La forme préférée est la notation "x:x:x:x:x:x:x:x", où "x" représente une valeur en hexadécimal des 8 portions de 16 bits de l'adresse. Les lettres peuvent être écrites en majuscules ou en minuscules.

Adresse IP statique/dynamique : On distingue deux situations pour assigner une adresse IP à un équipement :

  • Manière statique : adresse fixe et configurée manuellement.
  • Manière dynamique : l'adresse est automatiquement transmise et assignée grâce au protocole DHCP (Dynamic Host Configuration Protocol).

Structure d'une adresse IP : Une adresse IP est constituée de deux parties :

  • ID de réseau : les nombres à gauche.
  • ID d'hôte : les nombres à droite qui désignent les ordinateurs de ce réseau.

Pour déterminer les ID, on utilise les masques de sous-réseau sous forme de 4 octets, où 0 indique une annulation et 1 indique une conservation.

Exemple : Pour connaître l'adresse du réseau associée à l'adresse IP 34.208.123.12 avec le masque de réseau 255.0.0.0 : 

      00100010.11010000.01111011.00001100 (34.208.123.12)
    & 11111111.00000000.00000000.00000000 (255.0.0.0)
    = 00100010.00000000.00000000.00000000 (34.0.0.0)

Classes des réseaux :

Les réseaux TCP/IP se divisent en 3 classes pour faciliter la recherche d'un ordinateur sur le réseau.

  • Classe A : Le premier octet représente le réseau. Il y a 2^7 = 128 possibilités de réseaux. Les réseaux disponibles en classe A vont de 1.0.0.0 à 126.0.0.0. Les octets en vert représentent les ordinateurs du réseau, ce qui signifie que chaque réseau peut contenir 2^24-2=16,777,214 ordinateurs.
  • Classe B : Dans une adresse IP de classe B, les deux premiers octets représentent le réseau. Les deux premiers bits sont 1 et 0, ce qui donne 2^14 = 16,384 possibilités de réseaux. Les réseaux disponibles en classe B vont de 128.0.0.0 à 191.255.0.0. Les octets en vert représentent les ordinateurs du réseau. Ainsi, chaque réseau peut contenir 2^16-2=65,534 ordinateurs.
  • Classe C : Dans une adresse IP de classe C, les trois premiers octets représentent le réseau. Les trois premiers bits sont 1, 1 et 0, ce qui donne 2^21 = 2,097,152 possibilités de réseaux. Les réseaux disponibles en classe C vont de 192.0.0.0 à 223.255.255.0. L'octet de droite représente les ordinateurs du réseau, ce qui signifie que chaque réseau peut contenir 2^8-2=254 ordinateurs.
